# Drone Vision - Site de Filmagens Aéreas

Bem-vindo ao site da Drone Vision, especializada em filmagens aéreas com drones, edição de vídeo e desenvolvimento web.

## Estrutura do Projeto

```
Meusite/
├── index.html          # Página principal do site
├── css/
│   └── style.css      # Estilos do site
├── js/
│   └── script.js      # Funcionalidades JavaScript
├── img/               # Pasta para imagens (adicione suas imagens aqui)
└── videos/            # Pasta para vídeos (adicione seus vídeos aqui)
```

## Como Usar

1. **Personalização**:
   - Substitua `SEUNUMERO` no arquivo `js/script.js` pelo seu número de WhatsApp (apenas números, sem espaços ou caracteres especiais).
   - Adicione suas imagens na pasta `img/` e vídeos na pasta `videos/`.
   - Atualize as informações de contato na seção de rodapé do `index.html`.

2. **Personalização Avançada**:
   - Cores principais podem ser alteradas no arquivo `css/style.css` na seção `:root`.
   - Adicione mais seções ou serviços conforme necessário no `index.html`.

3. **Adicionando Vídeos**:
   - Para adicionar vídeos ao portfólio, substitua as divs com a classe `video-placeholder` no `index.html` por seus vídeos do YouTube, Vimeo ou arquivos de vídeo locais.
